# workspace

RootNote is a lightweight, ontology-free system for agents to store, connect, retrieve, and safely maintain knowledge in Markdown.

# design principles

When planning a change, work through these steps in order — proceed to each one only after the previous step is satisfied. 
1. Make requirements less dumb
2. Delete a part or process step
3. Simplify or optimize
4. Accelerate cycle time
5. Automate
